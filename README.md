# Qt 任务管理器

基于 PySide6 的现代化任务管理应用,支持普通任务、循环任务和常驻任务管理。

## 功能特性

### ✅ 普通任务
- 创建、编辑、删除任务
- 设置标题、描述、开始时间、截止时间
- 自定义提醒时间
- 状态管理(未完成/已完成)
- 标签分类
- 按提醒时间自动排序

### 🔄 循环任务
- 每天、每周、每月循环
- 自定义提醒时间
- 自动生成每日任务实例
- 新周期自动清理旧实例
- 支持设置结束日期

### 📌 常驻任务
- 备忘录式的长期任务
- 始终显示在侧边栏
- 快速查看重要事项
- 简单的标题和描述

### ⏰ 智能提醒
- 定时检查到期任务
- 强提醒弹窗
- 延迟提醒功能(Snooze)
- 系统托盘通知

### 🎨 其他特性
- 现代化 UI 设计
- 系统托盘常驻
- 多种筛选视图(今日/本周/循环/已完成)
- 右键菜单快捷操作

## 安装依赖

### 方式一:使用 pip(推荐)

```bash
# 进入项目目录
cd task_manager_qt

# 安装依赖
pip install -e .
```

### 方式二:仅安装 PySide6

```bash
pip install PySide6>=6.6.0
```

### 方式三:使用 requirements.txt

```bash
pip install -r requirements.txt
```

### 开发依赖(可选)

如果需要代码格式化和类型检查工具:

```bash
pip install -e ".[dev]"
```

## 运行应用

```bash
python main.py
```

或者如果使用 `pip install -e .` 安装,可以直接运行:

```bash

```

## 使用说明

### 创建普通任务
1. 点击 "➕ 新建任务" 按钮
2. 填写任务信息
3. 设置提醒时间
4. 点击保存

### 创建循环任务
1. 点击 "🔄 新建循环任务" 按钮
2. 填写任务标题和描述
3. 选择循环类型(每天/每周/每月)
4. 设置每日提醒时间
5. 如果选择"每周",需要勾选重复的日期
6. 可选设置结束日期
7. 点击保存

### 创建常驻任务
1. 双击左侧常驻任务区域
2. 填写任务标题和描述
3. 可选添加标签
4. 点击保存

### 任务操作
- **右键菜单**: 在任务上右键可进行编辑、删除、标记完成等操作
- **筛选视图**: 左侧边栏可切换不同的任务视图
- **系统托盘**: 最小化后可在系统托盘继续运行

## 数据存储

所有数据存储在 `tasks.db` SQLite 数据库中,自动创建在项目根目录。

## 注意事项

1. **循环任务清理**: 每天午夜会自动清理前一天的任务实例(不论是否完成)
2. **提醒检查**: 每分钟检查一次到期任务
3. **系统托盘**: 关闭窗口后程序继续在后台运行,可在系统托盘操作
4. **任务排序**: 任务列表按提醒时间排序,最早的在前,无提醒时间的在最后

## 技术栈

- **框架**: PySide6 (Qt 6)
- **数据库**: SQLite3
- **Python**: 3.10+
- **架构**: MVC 分层架构
